Smart Locker System

A secure and intelligent approach to personal storage using the CC3200 microcontroller

Project Overview

Our project is a secure access control system designed to authenticate users using a combination of a keypad and an RFID scanner. The system is built around the CC3200 microcontroller and features an OLED display for user interaction, a servo motor for door locking and unlocking, and a buzzer for audio feedback. The system aims to provide a standalone, interactive, and efficient method of entry authentication.

Secure Authentication

Uses PIN-based authentication through a keypad with masked input display and error handling for invalid attempts.

Fast Response

Quick unlock mechanism with automatic relocking after a set duration for added security.

AWS IoT Integration

Designed for real-time status notifications through cloud connectivity (partially implemented).

Design

Functional Specification

The access control system follows a structured state machine model:

  • Idle State: The system awaits user interaction. The OLED display prompts the user to enter a PIN or scan an RFID card.
  • Reading Input: The system reads the entered PIN via the keypad. If using a keypad, the display updates with masked input (e.g., "****").
  • Validating Input: The entered credentials are checked against the stored authentication database.
  • Access Granted: The OLED displays a success message, the buzzer sounds a confirmation tone, and the servo motor unlocks the door.
  • Access Denied: The OLED displays an error message, the buzzer plays an alert tone, and the system prompts the user to retry.

System Architecture

Our system comprises these core components:

  • CC3200 Microcontroller: The central processing unit that handles authentication and controls peripheral devices.
  • Keypad: Used to input the PIN code for authentication.
  • RFID Scanner: An alternative authentication method that scans RFID cards (partially implemented).
  • OLED Display: Provides real-time feedback to the user, displaying prompts and authentication status.
  • Servo Motor: Controls the locking and unlocking mechanism of the door.
  • Buzzer: Provides audio feedback to indicate success or failure of authentication.

Implementation

Hardware Implementation

The physical locking mechanism uses a servo motor with a custom-made lever system created from a metal pick. The keypad is connected to GPIO pins in a matrix configuration, while the OLED display operates via SPI protocol. We used four GPIO pins for keypad rows and four for columns, with PIN_64 dedicated to servo control.

Software Implementation

The firmware is structured around a state machine that handles authentication, locking/unlocking, and error conditions. We implemented keypad scanning with debounce filtering, secure PIN validation, and display buffer management for visual feedback. While we designed AWS IoT integration, this feature was only partially implemented in the final prototype.

Challenges & Solutions

Wi-Fi Connection Issues

Challenge: Getting the Wi-Fi module to connect properly to AWS services.

Solution: We attempted various approaches including firewall settings, security protocols, and power supply troubleshooting, but were only able to establish partial connectivity.

RFID Scanner Integration

Challenge: Getting consistent readings from the RFID scanner.

Solution: We tested different RFID modules and adjusted antenna positioning, but the RFID functionality remained only partially implemented.

Hardware Debugging

Challenge: Multiple issues including servo motor control, keypad input, and OLED display initialization.

Solution: We refined our PWM settings, improved debounce logic, and corrected wiring and library configurations to resolve most hardware issues.

Future Work

Several enhancements we'd like to implement in future iterations:

  • Fully implement AWS IoT and RFID authentication for remote monitoring and contactless access
  • Improve security features (encrypting PIN inputs, multi-factor authentication, failed login attempt lockout)
  • Enhance hardware efficiency (low-power mode, optimized servo response, electromagnetic lock upgrade)
  • Expand connectivity (mobile app/web dashboard, Bluetooth backup communication)
  • Integrate additional sensors (motion sensors, door open/close sensors, fingerprint scanner)
  • Improve user interface (enhanced OLED display feedback, LED status indicators)
